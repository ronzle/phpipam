<h2>phpipam</h2>

Container for PHPIPAM (http://phpipam.net/) 

<h3>Features</h3>

- configurable ssl support
- configurable mysql environment variables

<h3>Dependencies</h3>

Web Server

```
rlaborius\web
```

MySQL Server

```

```
